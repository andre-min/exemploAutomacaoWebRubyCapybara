# Sobre o Pojeto  
Este projeto tem  como objetivo automatizar teste web   

## requisitos:  
Node.js  
https://nodejs.org/en/   

Ruby + Devikit  
https://rubyinstaller.org/downloads/  

Cucumber.rb  
https://cucumber.io/docs/installation/  

## Tecnologias
- Ruby
- Capybara
- RSpec
- Cucumber
- Selenium webdriver

## Como executar 

Clonar o repositório, instalar as dependências  
`bundle install`

Executar testes de regressão  
```bundle exec cucumber```
